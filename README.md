# BIRTHDAY REMINDER

It is an application which eases the hassle of having to remember someone's birthdate.It also serves as a reminder.

## Installation

Install my-project with npm

```bash
  git clone
  npm install
  cd project
```

- React Hooks
- React Context API
- Styled components
